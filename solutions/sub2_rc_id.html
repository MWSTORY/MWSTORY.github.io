RemoteCombine
MWSTORY의 Remote Combine(Remote Management)솔루션 은 사용자 PC의 원격제어 기능 구현을 통해 보다 빠르고 확실한 고객서비스 및 각종 부가서비스를 실시간으로 제공할 수 있도록 한 원격제어 통합 솔루션입니다.
주요기능
접속환경에 구애받지 않는 안전한 원격제어 기술 방화벽, 공유기 사용중에도 제약 없이 접속 및 제어 가능 256Bit AES 기반의 보안시스템으로 안전성 보장
양방향(2-Way) 공유 기술을 통한 인터렉티브 환경 지원 양방향 화면 드로잉(Drawing) 기능 양방향 화면 캡쳐 및 공유 기능
완벽한 제어를 위한 다양한 부가 기능 시스템 정보 및 프로세스 정보 열람 기능 시스템간 파일 전송 / URL 전송 기능 (즐겨찾기 목록 관리) 편리한 환경 설정 / PC 전원 제어 기능
적용사례
원격제어
원격제어 서비스는 MWSTORY의 RMS 솔루션을 사용하여 전국 단위의 가맹점을 대상으로 서비스 제공자의 단말기/PC 사용 중 발생하는 고객 문의에 대한 온/오프라인 즉각 대응은 물론 실시간 보안 및 백업 서비스, 청소년 보호 서비스 등의 부가서비스를 제공하는 원격제어 통합 서비스입니다.
