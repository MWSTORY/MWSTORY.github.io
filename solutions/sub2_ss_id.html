SelfShot
SelfShot 은 증명, 반명함, 여권은 물론 웹서비스용 사진까지 편집해서 업로드 할 수 있는 어플입니다.
또한 이미 사진관에서 파일로 저장했던 사진파일이 있다면 간단한 편집으로 여러가지 증명사진을 만들어 사진관이나 집에서 인화하실수 있습니다.

주요기능
아이폰, 갤럭시 등의 다양한 스마트폰 운영체제 지원 ( iOS, Android, Windows Mobile ) 다양한 종류의 증명사진 제작 가능(2.5x3증명, 3x4 반명함, 3.5x4.5여권 등)
